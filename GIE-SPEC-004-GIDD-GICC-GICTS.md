## 1. Governance Integrity Data Dictionary (GIDD)
The **GIDD** provides canonical definitions and Protobuf schemas for all telemetry fields used by the Supervisory Control Plane (SCP).

### 1.1 Core Metric Definitions
- **G-SRI (Systemic Risk):** Bayesian synthesis of agent collusion and resonance drift.
- **CDI (Cognitive Dissonance Index):** Measures internal reasoning inconsistency within a TEE.
- **SAME (Secure Autonomous Machine Entity):** Stability metric for MoE router alignment.

## 3. Governance Integrity Control Tracking System (GICTS)
The **GICTS** monitors the real-time effectiveness of governance controls.

### 3.1 Effectiveness Metrics
- **Control Response Latency:** Time from breach detection to enforcement action (Target: < 200ms).
- **Attestation Coverage:** % of agentic reasoning steps anchored in a TEE (Target: 100%).
- **Verification Integrity:** % of ZK-proofs successfully aggregated by SnarkPack (Target: 100%).
